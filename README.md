## Introduction

This project is an advanced interactive Power BI dashboard built using a real-world dataset of global data job postings.

The dashboard helps users explore salary trends, job demand, remote work opportunities, degree requirements, health insurance availability, and country-wise job insights through an intuitive and interactive interface.

Compared to my previous Power BI project, this version introduces advanced Power BI features such as Dynamic Titles, KPI Cards, Bookmarks, Drill-through Pages, Custom Tooltips, Buttons, and Page Navigation to deliver a more professional analytical experience.

# Skills Showcased

This project demonstrates my progression from a basic Power BI dashboard to a more advanced, interactive report. Throughout this project, I explored powerful Power BI features used in real-world business dashboards.

- ⚙️ **Data Transformation (Power Query):** Cleaned, transformed, and prepared the raw dataset by handling missing values, standardizing schedule types, and creating custom columns for analysis.

- 🧮 **DAX Measures:** Created custom measures to calculate KPIs such as **Median Yearly Salary**, **Median Hourly Salary**, **Job Count**, and **Average Job Rating**.

- 📊 **Data Visualization:** Built a variety of charts including **Column Charts**, **Bar Charts**, **Line Charts**, **Area Charts**, **Scatter Charts**, **Stacked Charts**, **100% Stacked Charts**, **Cards**, **KPIs**, **Tables**, and **Matrix Visuals**.

- 🎨 **Dashboard Design:** Designed a clean and user-friendly dashboard with a consistent layout, rounded containers, dynamic visual titles, and improved readability.

- 📈 **Trend Analysis:** Visualized hiring trends over time and compared salaries, job counts, remote work opportunities, health insurance availability, and education requirements across different job roles.

- 🌍 **Comparative Analysis:** Compared salaries across countries, companies, job titles, and employment types to uncover meaningful insights from the global data job market.

- 🧭 **Interactive Reporting:**
  - **Slicers:** Dynamic filtering by Job Title and other dimensions.
  - **Buttons & Page Navigation:** Seamless navigation between report pages.
  - **Bookmarks:** Interactive navigation without leaving the current report.
  - **Drill-through Pages:** Detailed analysis for individual job roles.
  - **Dynamic Titles:** Chart titles automatically update based on user selections.
  - **Custom Tooltips:** Display additional contextual information when hovering over visuals.

- 📋 **Advanced Reporting Features:**
  - KPI Cards for quick business insights.
  - Conditional formatting for improved readability.
  - Matrix with Sparklines to visualize trends.
  - Interactive Tables with sorting and filtering.
  - Top N filtering for focused analysis.
  - Cross-filtering and cross-highlighting between visuals.

- 🚀 **Business Intelligence Skills:** Applied real-world dashboard design principles to transform raw job posting data into an interactive decision-support dashboard suitable for recruiters, job seekers, and data professionals.

# Dashboard Overview

This report is organized into **two interactive dashboard pages**, providing both a high-level overview of the data job market and a detailed job-specific analysis.

---

## 🏠 Page 1: Data Jobs Dashboard

![Dashboard Overview](images/dashboard-page1.gif)

The main dashboard provides a comprehensive overview of the global data job market. Users can quickly explore salary trends, hiring demand, and job distribution through interactive visuals.

### Highlights

- 📊 KPI Cards showing:
  - Total Job Postings
  - Median Yearly Salary
  - Median Hourly Salary
  - Average Job Rating

- 📈 Hiring Trend Analysis using a line chart.

- 💰 Salary Comparison with scatter charts.

- 📊 Job Counts ranked by Job Title.

- 📋 Interactive Matrix displaying:
  - Job Count
  - Median Yearly Salary
  - Median Hourly Salary
  - Trend Sparklines

- 🎛️ Dynamic filtering using slicers.

- 📝 Dynamic chart titles that automatically update based on user selections.

- 🔗 Drill-through button for navigating to detailed job-specific analysis.

---

## 🔍 Page 2: Job Title Drill-Through

![Job Drill Through](images/dashboard-page2.gif)

The drill-through page allows users to explore detailed insights for an individual job title selected from the main dashboard.

### Highlights

- 💵 Median Yearly Salary

- ⏰ Median Hourly Salary

- 🏠 Work From Home Percentage

- 🎓 No Degree Requirement Percentage

- 🩺 Health Insurance Availability

- 🌍 Geographic distribution of job postings using Map visuals.

- 🏢 Top hiring platforms for the selected role.

- 📅 Employment Schedule Type breakdown using a Treemap.

- 🔄 Interactive navigation back to the main dashboard using buttons.
